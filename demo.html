Demo Scenes Reconstructed from a Single-view Image using VistaDream

Main Page

* Note 1: Here, we present only the **heavily downsampled point clouds** for efficient display and interaction.
It might take a while to load the scene.

* Note 2: Mouse: ''Drag with left click'' to rotate view || ''Scroll''' to zoom in/out || ''Drag with right click'' to move view
Keyboard: ''W'' - Move forward || ''S'' - Move backward || ''A'' - Move left || ''D'' - Move right

* Note 3: Check / Uncheck the ''Remove Input View'' below will Gray-Out / Retain the area of input view.
